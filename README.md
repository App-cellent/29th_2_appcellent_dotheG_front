# Frontend

## ✨Main 기능
- 회원가입/로그인
- 튜토리얼 제공
- 만보기 측정
- 친환경 활동 인증
- 데일리 환경 퀴즈
- 캐릭터 뽑기 및 수집
- 주간/월간 성과보고서
- 마이페이지
